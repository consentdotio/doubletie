## 🏗️ Development

Prerequisites:
- Node.js >= 18
- pnpm >= 8

```bash
# Clone repository
git clone https://github.com/consentdotio/doubletie.git
cd c15t

# Install dependencies
pnpm install

# Start development
pnpm dev

# Run tests
pnpm test
```

## 🧪 Testing

We use Vitest and Playwright for testing:

```bash
# Run unit tests
pnpm test:unit

# Run E2E tests
pnpm test:e2e

# Run all tests
pnpm test
```
